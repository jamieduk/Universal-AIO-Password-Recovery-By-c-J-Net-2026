## Overview

Universal-AIO-Password-Recovery is a unified recovery framework designed to handle multiple encrypted file types using wordlist-based password recovery.

It provides a single interface for:

- ZIP / RAR / 7z archives  
- PDF documents  
- MS Office files  
- OpenDocument formats  
- Disk images and containers  
- OpenSSL encrypted files  
- GPG / PGP encrypted data  
- Private keys and certificates  
- Databases and backups  
- eBooks and other encrypted formats
